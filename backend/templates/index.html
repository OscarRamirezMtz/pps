Descripción del proyecto

Proyecto final de la Experiencia Educativa Programación Segura, plataforma web que permita administrar servicios de un conjunto de servidores Linux.

En este sistema, un administrador puede registrar servidores, una vez se tiene un servidor registrado, es posible levantar, administrar y monitorizar servicios instalados en el servidor.

Requisitos Funcionales

  • Loguearse como sysadmin para acceder al sistema.
  • Dar de alta servidores con información detallada.
  • Configurar servicios en cada servidor.
  • Ejecutar respaldos de manera programada.
  • Monitorizar servicios.
